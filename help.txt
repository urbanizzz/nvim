Работа с буферами и историей перемещений
Просмотр и навигация
  :ls — показать список всех открытых буферов (файлов в памяти).
  :b N — Перейти в буфер под номером N (например, :b 3).
  :bn (:bnext) — Перейти к следующему буферу в списке.
  :bp (:bprevious) — Перейти к предыдущему буферу в списке.
  ctrl + ^ (или ctrl + 6) — переключиться на предыдущий активный файл (альтернативный буфер #).
История перемещений (Jumplist)
  Ctrl + O — Вернуться назад по истории курсора (к предыдущему файлу или строке).
  Ctrl + I — Перейти вперед по истории курсора.
Закрытие и очистка
  :bd (:bdelete) — Закрыть текущий буфер (убрать файл с экрана и из памяти).
  :bd N — Закрыть буфер под конкретным номером N.
  Ctrl-W + c - закрытие текущего окна/сплита.
  Ctrl-W + q - выход из текущего окна.
TypeChecker:
  <C-w>w — циклически переключает курсор между всеми открытыми окнами.
  <C-w>j — перейти в окно ниже.
  <C-w>k — перейти в окно выше.
  :cclose - закрыть окно typechecker
